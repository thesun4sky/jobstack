# 네이버 백엔드 개발자 채용공고

## 회사 소개
네이버는 대한민국 최대 인터넷 기업으로, 검색, 커머스, 핀테크, 콘텐츠, 클라우드 등 다양한 서비스를 운영하고 있습니다.

## 모집 부문
- 직무: 백엔드 개발자 (서버 플랫폼)
- 고용형태: 정규직
- 근무지: 경기도 성남시 분당구

## 자격요건
- Java/Kotlin 기반 서버 개발 경험
- Spring Framework 활용 경험
- RDBMS 및 NoSQL 사용 경험
- RESTful API 설계 및 구현 경험

## 우대사항
- 대용량 트래픽 처리 경험
- MSA(마이크로서비스 아키텍처) 경험
- Kubernetes, Docker 활용 경험
- 오픈소스 기여 경험
- 기술 블로그 운영
