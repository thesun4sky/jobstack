### 클래스 소개

취업 준비에 필요한 건 많습니다.
기업 분석, 이력서, 자소서, 포트폴리오, 모의면접, 지원 관리, 회고...
**이걸 혼자 다 하려면 3개월은 잡아야 합니다.**

근데 만약, 이 과정을 **AI 에이전트 8명이 팀으로 코칭**해준다면요?

**jobstack**은 Claude Code 위에서 돌아가는 오픈소스 취업 코칭 시스템입니다.
폴더에 이력서 하나 넣고 `/auto` 치면, AI가 파일을 감지하고
"이력서 첨삭부터 할까요, 기업분석부터 할까요?" 물어봅니다.

그냥 따라가기만 하면 됩니다. **AI가 단계를 설계하고, 실행하고, 피드백**합니다.

핵심은 이겁니다:
- 자소서 넣으면 → **7가지 탈락 패턴 자동 진단** + 점수 매기기
- 채용공고 넣으면 → **7가지 소스에서 키워드 자동 추출** + 반영률 체크
- "결이요" 프레임워크로 → **자소서 자동 재구성** (Before/After)
- 면접 예상 질문 → **25~32개 자동 생성** + 모의면접 시뮬레이션

9년간의 실전 첨삭 노하우가 **프롬프트에 통째로** 들어가 있습니다.
사람 코치 퀄리티를 AI가 **무제한, 무료로** 해주는 겁니다.

**코딩 몰라도 됩니다.** 설치 3분, 사용법은 슬래시(/) 하나.
이 특강에서 라이브로 설치하고, 본인 자소서로 직접 돌려봅니다.

### 추천 대상

- 자소서/이력서 첨삭에 **돈 쓰기 부담스러운** 분 (jobstack은 무료 오픈소스)
- 취업 준비를 **어디서부터 시작해야 할지 모르겠는** 분 (AI가 단계를 안내)
- 기업별로 자소서를 **매번 새로 쓰는 게 힘든** 분 (키워드 체크리스트 자동 생성)
- 모의면접을 **혼자 연습할 방법이 없는** 분 (AI 면접관이 질문 + 피드백)
- AI 도구를 취업 준비에 **제대로 활용**하고 싶은 분
- "ChatGPT로 자소서 써봤는데 **너무 뻔하게 나온다**"는 분 (jobstack은 9년 실전 노하우 기반)
