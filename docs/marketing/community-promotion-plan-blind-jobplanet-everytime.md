# 취업 커뮤니티 홍보 계획 (블라인드 · 잡플래닛 · 에브리타임)

**목표**: JobStack 웹 SaaS 인지도·가입·Free→Pro 전환  
**톤**: 과장 금지, **체험 근거·방법론(결이요)·13 스킬** 중심, 광고 느낌 최소화

## 공통 원칙

1. **플랫폼 규칙 우선**: 각 앱 게시판·광고·스팸 정책 확인 후 게시. 불확실하면 “정보 공유” 형태로 최소한의 링크·문의 유도.
2. **가치 우선**: 팁·체크리스트·실패담·비포에프터(개인정보 비식별)로 시작하고, 자연스럽게 도구 소개.
3. **CTA**: “무료로 ○○까지 해봤다” → 공식 랜딩/가입(UTM 파라미터로 채널 구분).

**UTM 예시**: `?utm_source=blind&utm_medium=community&utm_campaign=launch_q2`

---

## 1. 블라인드 (Blind)

**특성**: 직장·취업 토론, 익명, IT·대기업 관심 높음.

| 항목 | 내용 |
|------|------|
| **타겟 보드** | 취업·이직, 주니어, 대기업/IT 취준 (규정상 허용되는 곳만) |
| **포맷** | (1) 자소서 쓰는 시간 줄인 경험 (2) GPT만 쓸 때와 워크플로 비교 (3) 질문 유도 |
| **금지·주의** | 명백한 홍보만 하는 글, 링크 도배, 경쟁사 비방 |
| **KPI** | 게시글 저장·댓글 수, UTM 유입, 가입 |

**실행 주기**: 월 1~2회 **고품질** 글(양보다 질).

---

## 2. 잡플래닛 (Jobplanet)

**특성**: 기업 리뷰 중심, 취업 정보 검색 유입.

| 항목 | 내용 |
|------|------|
| **타겟** | 면접·자소서 키워드 검색자, 기업별 준비 글 |
| **포맷** | 기업 준비 **노하우 요약** + “정리용으로 쓰는 도구” 한 줄(과장 X) |
| **금지·주의** | 리뷰 조작·허위 경험 금지, 커뮤니티 가이드 준수 |
| **KPI** | 게시글 조회·반응, UTM |

**실행 주기**: 분기별 또는 채용 시즌 전 **2~3건** 집중.

---

## 3. 에브리타임 (Everytime)

**특성**: 대학교 단위, 동아리·수업·취업 게시판.

| 항목 | 내용 |
|------|------|
| **타겟** | 취업·인턴 게시판, 학과·학년별 커뮤니티 |
| **포맷** | 선배 후배 톤 **후기형**(시간 절약, 자소서 구조 잡는 법), 필요 시 동아리 오프라인 설명회 연계 |
| **금지·주의** | 학교별 홍보 규정, 상업 게시 제한 시 동아리 협찬·캠퍼스 앰버서더 검토 |
| **KPI** | 댓글·DM 문의, UTM, 캠퍼스별 가입 |

**실행 주기**: 개강·채용 시즌 **집중 2주 캠페인** + 학기 중 1회 유지.

---

## 4. 콘텐츠 뱅크 (재사용)

- **핵싱 메시지**: “자소서 한 통 돌리기”가 아니라 **전 과정이 이어지는지**.
- **근거**: 4년 60건+ 검증, 13 스킬(구체 스킬명 3~4개만 나열).
- **자산**: 스크린샷(민감정보 제거), 30초 데모 영상(랜딩·SNS 공용).
